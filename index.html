‚ùì Frequently Asked Questions

Is Unblocked free to use?

During beta, Unblocked is completely free! We're gathering feedback to build the best product possible. Pricing for the full release will be announced later.

How does data sync work?

Unblocked uses Firebase for real-time synchronization. Your data is stored in the cloud and syncs automatically across all your devices. Changes appear instantly!

Can I use Unblocked offline?

Basic functionality works offline using local storage. Your changes will sync to Firebase once you're back online.

Is my data private and secure?

Currently in beta testing mode, data is stored in Firebase with test-mode security rules. Full authentication and private boards are coming in the next release!

Can I collaborate with others?

Board sharing and collaboration features are in development! For now, anyone with the board URL can view and edit cards (beta testing mode).

How do I add Unblocked to my iPhone/iPad?

Open this site in Safari, tap the Share button, then "Add to Home Screen". It will work like a native app!

Can I export my data?

Yes! Use the Export button in the board to download all your cards as a JSON file. Import functionality lets you restore or share boards.
